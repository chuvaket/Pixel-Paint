# ENG

To run, download and run the exe file, or paste the python code into your editor and run it.

The available <b><i>colors</i></b> are: <br />
<b><i>red</i></b> <br />
<b><i>yellow</i></b> <br />
<b><i>blue</i></b> <br />
<b><i>cyan</i></b> <br />
<b><i>white</i></b> <br />
<b><i>black</i></b> <br />
<b><i>green</i></b> <br />
<b><i>magenta</i></b> <br />

You can also use the <b><i>"fill"</i></b> command to fill a specific area.<br />
Example: <code>"Enter pixel(x y color): fill red"</code> --> <code>"Specify fill borders(x x y y) (press Enter to fill all): 1 3 2 5"</code> --> fills the area with red color from pixel 1 to pixel 3 horizontally and from pixel 2 to pixel 5 vertically. <br />
Note: The <b><i>"fill"</i></b> command fills the area without preserving the pixels that you have already filled in the area.<br />

If you need to exit the program, you can do it directly in the console using the <b><i>"exit"</i></b> command.
Example: <code>"Enter pixel(x y color): exit"</code>



# РУС/RUS

Для запуска скачайте и запустите exe-файл или перенесите python код в свой редактор и запустите.

<b><i>Цвета</i></b> доступные к использованию: <br />
<b><i>red</i></b> <br />
<b><i>yellow</i></b> <br />
<b><i>blue</i></b> <br />
<b><i>cyan</i></b> <br />
<b><i>white</i></b> <br />
<b><i>black</i></b> <br />
<b><i>green</i></b> <br />
<b><i>magenta</i></b> <br />

Также вы можете использовать команду <b><i>"fill"</i></b> для заполнения определенного пространства.<br />
Пример: <code>"Запишите пиксель(x y color): fill red"</code> --> <code>"Укажите границы заливки(x x y y) (нажмите Enter - закрасить все): 1 3 2 5"</code> --> закрасит область красным цветом с пикселя 1 по пиксель 3 по горизонтали и с пикселя 2 по пиксель 5 по вертикали.<br />
Примечание: команда <b><i>"fill"</i></b> закрашивает пространство не сохраняя уже закрашенные вами пиксели на это области.<br />


Если вам необходимо выйти из прграммы, то можно сделать это прямо в консоли, использовав команду <b><i>"exit"</i></b>.
Пример: <code>"Запишите пиксель(x y color): exit"</code>
